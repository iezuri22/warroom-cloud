One-time setup
You need two Shortcuts: Deep Work (turn on) and Deep Work Off (turn off, optional). Start passes the selected minutes as input.
Show setup steps
  1. iPhone — Settings → Focus → +: create "Deep Work". Pick which apps/people are allowed to notify you.
  2. iPhone — Settings → Focus → Share Across Devices: on. Same toggle on Mac → System Settings → Focus.
  3. iPhone — Shortcuts app → +: rename it Deep Work.
    Action 1: Receive Shortcut Input → accept Text. Provide default "60" if nothing passed.
    Action 2: Set Focus → Deep Work → Turn On → For X minutes — set X to Shortcut Input (the variable).
    Save.
  4. Optional: create a second shortcut Deep Work Off with one action: Set Focus → Deep Work → Turn Off. Used when you tap End Early / Mark Done.
  5. Shortcuts app → Settings → iCloud Sync: on. Both shortcuts now sync to your Mac + iPad automatically.
  6. Mac: set Deep Work's app rules for Mac (System Settings → Focus → Deep Work → Allowed Apps/People).
Test it: tap Start Focus above → Safari prompts "Open in Shortcuts?" → tap Open. All your devices enter Deep Work.
